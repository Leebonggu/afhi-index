# AFHI-INDEX
### 미래예측에 필요한 경제 데이터를들 한 곳에 모아서 볼 수 있게 해주는 페이지
### FRED에서 제공해주는 데이터를 기반으로 미국 지표를 주제별로 모아 보여준다.

### 섹션
- 미국 경제 전반
- 미국 국채(10년물): 국채가격, 국채수요공급 관련
- WTI Crude Oil Price
- 미국 선행지표


### 작업 로그
